## 快速开始

1. 使用以下命令克隆本仓库：

```bash
git clone https://github.com/MK2112/nn-zero-to-hero-notes.git
```

2. 进入仓库目录，确保运行 `pip install -r requirements.txt`。
3. 运行以下命令启动 Jupyter Notebook：

```bash
jupyter notebook
```

4. 在 Jupyter 中导航到仓库目录，打开所需的 notebook。

[通用内容](#通用内容)和[演讲](#演讲)笔记使用 Markdown 格式编写，而非 Jupyter Notebook。
Markdown 笔记同样可以在 GitHub 上查看。如需最佳的离线 Markdown 阅读体验，建议使用 [Obsidian](https://obsidian.md)。
